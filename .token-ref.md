# Token Storage Reference

Tokens are stored outside the repo at:
`~/.config/opencode/secrets/`

| File | Token |
|------|-------|
| `github_pat.txt` | GitHub PAT (fine-grained, Actions secrets: Write) |
| `cloudflare_token.txt` | Cloudflare API Token (Pages: Edit) |
| `pythonanywhere_token.txt` | PythonAnywhere API Token |
| `README.md` | Full reference index |

## Current Status

- `CLOUDFLARE_API_TOKEN` → ✅ Set as GitHub repo secret (used by CI/CD deploy workflow)
- PythonAnywhere Token → Used by `backend/scripts/deploy-backend.sh`

## Deploy Commands

- **Frontend:** Push to `main` → auto-deploys to Cloudflare Pages
- **Backend:** `bash backend/scripts/deploy-backend.sh` (run from PythonAnywhere Bash console)
